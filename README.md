# SwitchZ

SwitchZ is a NeoForge client mod for Minecraft 1.21.1 that lets you switch offline usernames without restarting the game client.

## Development

Use the Gradle wrapper from the repository root:

```sh
./gradlew build
./gradlew runClient
```

If your local Gradle cache gets out of sync, refresh dependencies with:

```sh
./gradlew --refresh-dependencies
```

The project targets Java 21 and NeoForge 21.1.x.
